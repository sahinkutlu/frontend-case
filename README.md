# Multi Select <a name="#about-project"></a>

**Multi Select** is a React-based multi-select autocomplete component that integrates with the "Rick and Morty" API to search and select characters.

### Key Features <a name="key-features"></a>

- **Autocomplete Search**: Implement a search input that queries the "Rick and Morty" API based on user input.
- **Character Display**: Display character information in the dropdown, including character image, name, and number of episodes played.
- **Query Highlighting**: Highlight the search query in the displayed results for better visibility.
- **Multi-Select Functionality**: Enable the user to select multiple characters from the dropdown and display them below the input field.
